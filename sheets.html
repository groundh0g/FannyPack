About FannyPack  

FannyPack is a spritesheet tool written in HTML & JavaScript by @groundh0g.

Original concept and inspiration from PixiTexturePacker by @SebastianNette, although none of his code was used on this project. Sebastian was inspired by the excellent TexturePacker software, by Andreas Löw. I personally own a TexturePacker license, and highly recommend it.

This project uses modules from the following opensource and public domain projects: Blob, Bootstrap, FileSaver, jQuery, json2, jszip, object-keys.js, and UUID.
