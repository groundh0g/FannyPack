About FannyPack

FannyPack is a spritesheet tool written in HTML & JavaScript by @groundh0g.

Original concept and inspiration from PixiTexturePacker by @SebastianNette.

Changes for FannyPack include UI redesign, removal of dependencies, addition of features, other tweaks -- like getting the darn thing to work in Chrome. :)
